# Laboration 3 + 4 + 5 Sass, Animation, Diagram och Kartor

## 🔗 https://sass-lab.onrender.com/
* Detta projekt är en del av kursen Frontend-baserad webbutveckling.

---
### 🎨 Laboration 3: Arbetsprocess & SASS

* Använda en automatiserad arbetsprocess
* Skriva och implementera SASS-filer med automatisk konvertering
* Ha stöd för ljust och mörkt tema

--- 
### ✨ Laboration 4: Animationer

På sidan för animationer finns:
* Animation på rubrik, textfärg + skugga
* Animation för prick som rör sig fram och tillbaka på sidan samt byter färg
* Streck som dras
* Knapp som blir bredare vid hovring
* Bild som studsar när knappen klickas

---

### 📊 Tillägg för Laboration 5, Diagram och Kartor

* Använt Chart.js för att:
* Rendera ett **stapeldiagram** över de 6 mest sökta kurserna
* Rendera ett **cirkeldiagram** över de 5 mest sökta programmen 

* Interaktiv karta med Leaflet & Nominatim:
* Interaktiv karta byggd med **Leaflet**
* Integrerad sökfunktion som använder **Nominatim API**
* Vid sökning flyttas kartans vy till de nya koordinaterna.
